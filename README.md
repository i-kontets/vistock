# VISTOCK

## 概要
VISTOCK は、冷蔵庫内の食材を管理し、  
賞味期限切れや無駄買いを減らすことを目的とした個人開発アプリです。

スマホアプリをメインとし、  
後々 Web 版での可視化・分析や AI を活用した入力補助を追加予定です。

---

## コンセプト
- 冷蔵庫の中を「見える化（view Stock）」
- スマホでサクッと入力
- 期限切れ前に通知
- 将来的に入力を自動化（レシート分析）
- 作り切れるスコープで段階的に拡張

---

## 構成

### スマホアプリ（メイン）
- React Native（Expo）
- iOS 対応

### Web（サブ）
- React
- 在庫・履歴の可視化用

### バックエンド
- PHP
- MySQL
- 共通 API としてスマホ / Web から利用

### 本番環境
- ロリポップ（PHP / MySQL）
- Web は build した静的ファイルを配置

---

## 主な機能

### スマホアプリ
- 食材の手動登録
  - 商品名
  - カテゴリ
  - 数量
  - 賞味期限 or 購入日
- 食材一覧表示
- 消費・削除
- 賞味期限前のローカル通知
- 買い物リスト
- （将来）カメラによるレシート撮影
  - AIによるレシート分析で登録のプロセスを簡易化
- （将来）位置情報からその場所用の買い物メモの表示・通知

### Web
- 食材一覧の確認
- カテゴリ別在庫数の表示
- 期限切れ・期限間近の食材確認
- 消費・廃棄の履歴確認

---

## スマホ機能の利用（Expo）

- カメラ（レシート撮影用）
- ローカル通知（期限通知）
- 位置情報（将来）
  - スーパー到着時に買い物メモを通知
- 生体認証（将来検討）

※ ネイティブコードは使用しない  
※ Dynamic Island / 常時バックグラウンド処理は対象外
